ICON INSTRUCTIONS
=================

Untuk extension ini, Anda memerlukan 3 ukuran icon dalam format PNG:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

CARA MEMBUAT ICON:

Opsi 1 - Menggunakan SVG yang sudah disediakan:
1. Buka file icon.svg di folder ini
2. Gunakan tool online seperti:
   - https://cloudconvert.com/svg-to-png
   - https://convertio.co/svg-png/
3. Convert ke PNG dengan ukuran yang dibutuhkan (16x16, 48x48, 128x128)
4. Simpan dengan nama: icon16.png, icon48.png, icon128.png

Opsi 2 - Membuat icon sendiri:
1. Gunakan tool design seperti Canva, Figma, atau Photoshop
2. Buat design dengan tema robot/AI/image generator
3. Export dalam 3 ukuran: 16x16, 48x48, dan 128x128 pixels
4. Simpan sebagai PNG dengan nama yang sesuai

Opsi 3 - Menggunakan icon placeholder:
Jika Anda ingin langsung mencoba extension tanpa icon khusus,
Anda bisa menggunakan icon emoji atau icon sederhana untuk sementara.

CATATAN:
Icon-icon ini akan muncul di:
- Toolbar Chrome (icon16.png)
- Extension management page (icon48.png)
- Chrome Web Store (icon128.png)
